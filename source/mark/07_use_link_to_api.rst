=============
将链接指向API
=============

如果你想要在你的文档中使用链接指向API，你可以使用 ``:class:`~net_inspect.NetInspect`` 格式。这将会在文档中生成一个链接，指向 ``NetInspect`` 类的文档。

同样的可以使用 ``:obj:``, ``:meth:``, ``:func:``, ``:attr:``, ``:exc:``, ``:mod:``, ``:data:`` 等来指向其他对象。
